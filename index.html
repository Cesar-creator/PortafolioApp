¿Por qué Contratarme?

Siempre me ha apasionado el desarrollo de aplicaciones Web desde mis tiempos escolares, profundizando durante mi tiempo en la Universidad por medio de aprendizaje autónomo, he conseguido conocimientos y habilidades tanto teoricas como prácticas para el desarrollo de software con el uso de algunas herramientas o tecnologías como las siguientes:

Javascript

Lenguaje de programación orientado para las aplicaciones Web y su interactividad.

Reactjs

Libreria de Javascript diseñada para crear interfaces de usuario en el desarrollo de aplicaciones en una sola página.

Git Terminal

Software de control de versiones para aplicaciones cuando estas tienden actualizarse o su codigo fuente es robusto. Además, para trabajar en diversas ramas de proyectos.

Sass

Lenguaje de hoja de estilos en cascada, siendo un procesador de CSS permite generar de manera automática, hojas de estilo y caracteristicas adicionales.

Mongo Database

Sistemas para la gestión de datos NoSQL o no relacional. Orientado a documentos que se almacenan en formato BSON, una representación binaria de JSON.

Firebase

Es una plataforma en la nube creada por Google para el desarrollo de aplicaciones web y movil.
